Ajax

W3school Ajax简介

https://www.w3school.com.cn/js/js_ajax_intro.asp

简介

  • Ajax(Asynchronous Javascript And XML):异步的JavaScript 和 XML

  • Ajax并非编程语言，它为作用仅有两点：

    1. 与服务器进行数据交换：通过AJAX向服务器发送请求(XMLHttpRequest 对象)，获得服务器响应数据。
    2. 异步交互：可以在不重新加载整个页面的情况下，于服务器交换数据并更新部分网页的技术。

简介

Axios 是一个基于 promise 网络请求库，作用于node.js 和浏览器中。 它是 isomorphic 的(即同一套代码可以运行在浏览器和node.js中)。在服务端它使用原生 node.js http 模块, 而在客户端 (浏览端) 则使用 XMLHttpRequests。 ——复制于官网

Axios 请求方式别名

  • 为了方便起见，Axios已经为所有支持的请求方式提供了别名。

axios.request(config)

axios.get(url[, config])

axios.delete(url[, config])

axios.head(url[, config])

axios.options(url[, config])

axios.post(url[, data[, config]])

axios.put(url[, data[, config]])

axios.patch(url[, data[, config]])

注意

在使用别名方法时， url、method、data 这些属性都不必在配置中指定。
