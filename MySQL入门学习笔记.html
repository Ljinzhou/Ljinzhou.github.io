4.DDL&DQL&DML&DCL

SQL程序语言的四种类型，对数据库的基本操作都属于这四类，分别为：数据定义语言(DDL)、数据查询语言（DQL）、数据操纵语言（DML）、数据控制语言（DCL）

4.1.DDL

数据定义语言(DDL)，是对数据库和表的增删改查。

查询数据库：

语法：show databases;

（不要忘记结尾的分号）

使用数据库：

语法：use {数据库名称};

查看当前正在使用的数据库：

语法：select database();

创建数据库：

语法：create database 数据库名称;

带判断条件创建数据库：（推荐使用）

语法：create database if not exists 数据库名称;

（没有就创建）

删除数据库：

语法：drop database 数据库名称；

带判断条件删除数据库：（推荐使用）
